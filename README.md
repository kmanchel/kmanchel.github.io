# Personal Website

This is the source code for my personal website hosted at [kmanchel.github.io](https://kmanchel.github.io).

## Built With

- Jekyll (GitHub Pages)
- Minima theme

## Customization

- Edit `_config.yml` to update site settings (title, email, description, etc.)
- Edit `index.md` to update the homepage
- Edit `about.md` to update the about page
- Add new pages by creating `.md` files with front matter

## Local Development (Optional)

If you want to preview the site locally:

1. Install Ruby 2.7 or higher
2. Run `bundle install`
3. Run `bundle exec jekyll serve`
4. Visit `http://localhost:4000`

## Deployment

Push changes to the `main` branch and GitHub Pages will automatically build and deploy your site.
